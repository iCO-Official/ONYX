Как получить ключ

Apple previews работают без ключа. Для SoundCloud есть встроенный client_id по умолчанию, но вы можете указать свой в любое время. Для Jamendo нужен client_id, для YouTube используйте YouTube Data API key. ONYX показывает результаты поиска и внешние ссылки, но не реализует обход защиты или нелицензионный доступ к потокам.
